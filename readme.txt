Disables the Wordpress "feature" that injects paragraph tags into double line breaks.

== Description ==

With this plugin installed, you can say goodby to automatic and unwanted paragraph tags, your markup can stay beautiful, and never again will you have to wonder why there are line breaks where none should be.

Github: https://github.com/binaryorganic/Disable-Automatic-P-Tags

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `remove-wpautop.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
